# Review Rubric

Severity guidance:

- Critical: game cannot start, crashes, data loss, commands unsafe, or mode cannot be completed.
- High: win/loss rules wrong, controls broken, powerup cleanup corrupts match state, tests missing for risky shared rules.
- Medium: UI confusing, audio/visual feedback misleading, architecture boundary erosion, incomplete settings behavior.
- Low: polish issues, naming clarity, minor duplication, non-blocking documentation drift.

Review output should lead with findings. Include file references and concrete fixes.
